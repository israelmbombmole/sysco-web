SYSCO Web — documentation figures
=================================

Single complete user manual
---------------------------

The **one-book** user guide is:

  docs/00-User-Manual-Complete-Book.md

It references the PNGs below. **Placeholder images** (coloured bars + labels) are generated by:

  python docs/figures/generate_placeholder_pngs.py

Replace them with **real screenshots** from your staging environment, then add **numbered red arrows**
in an image editor (Paint.NET, GIMP, PowerPoint export, etc.) so they match the step numbers in the manual.

Markdown image paths
--------------------

From files under `docs/`, use:

  ![caption](figures/fig-XX-....png)

Pandoc resolves these relative to the directory of the first input file (`docs/`), so run exports from the
repository root with:

  pandoc docs/00-User-Manual-Complete-Book.md -o SYSCO-Web-User-Manual-Complete.pdf ...

Regenerating the single Markdown book
-------------------------------------

After editing any `02-` … `17-` user manual source file:

  python docs/build-complete-user-manual.py
